## 1) OVERALL SHELL

- Full-screen dark app, RTL Arabic
- Root wrapper class concept: frosted glass over wallpaper
- Background layer BEHIND everything: full-screen wallpaper image /MAN.png (cover center) with optional owner custom image
- On top: semi-transparent glass panels, green glow accents
- Font: clean Arabic sans, small bold labels (10px–12px), black weight headings

COLORS (exact):
- Page bg: #060a12 / #050806
- Accent primary: #10b981 (emerald green) — NOT cyan/teal
- Accent secondary: #a3e635 (lime)
- Text: #f8fafc, muted #94a3b8
- Glass: rgba(255,255,255,0.04) fill, rgba(255,255,255,0.08) border
- VIP gold: #eab308 (only for VIP badges, not whole panels)
- Borders on columns: border-white/8, subtle green border-green-500/10 on hover

## 2) TOP HEADER (fixed ~64px — REQUIRED)

Full width bar, blur backdrop, always visible.

RTL layout inside header:

RIGHT side (start in RTL):
- Circular logo SVG "AMLogo" ~44px — green calligraphy Lamma mark with glow + small crown on top
- Next to logo text block:
  - "LAMMA CHAT" white bold small caps

  - subtitle "بالصحبة الأحلى" gray small

## 3) DESKTOP BODY — 3 COLUMNS + COLLAPSE HANDLES

Below header: flex row, 3 columns. CENTER chat is WIDEST (hero). Side columns ~300–340px each.

### Column order in RTL (what user sees):
| Visual position | Content |
|----------------|---------|
| RIGHT strip | الغرف + المتصلون |
| CENTER (widest) | الشات |
| LEFT strip | VIP teaser + راديو + DJ/موسيقى |

### COLLAPSE BUTTONS (الغوامين — CRITICAL, missing in wrong builds):

On desktop xl+ ONLY, two vertical tab buttons fixed at vertical center of screen edges:
- One on the edge between LEFT column and center: height ~64px, width ~28px, rounded toward center, dark bg black/35, green border, chevron icon
  - Click toggles LEFT column (store/radio/music): collapses to width 0 with opacity 0 transition 300ms
  - When collapsed, chat column expands wider
- One on the edge between RIGHT column and center: same style, toggles RIGHT column (rooms/members)

When either sidebar hidden, center chat gets border accent on that side.

### RIGHT column — two STACKED glass sections (NOT one list):

Glass card rounded-3xl (lamma-glass), split by DRAGGABLE horizontal divider (12px hit area, fire/glow divider style):
1. TOP section (~62% default): "الغرف" with BookOpen icon
   - Rooms grouped by category headers: اجتماعي، مجتمع، ترفيه، علاقات، خاصة
   - Each category: icon + name + room buttons (flag emoji + name + online count pill)
   - Active room: green tint bg rgba(16,185,129,0.10), green border
   - Button "إنشاء غرفة" green outline at top (if allowed)
2. BOTTOM section (~38%): "المتصلون" online members list
   - Each member: colored nickname, role chips (BOSS/ADMIN/VIP/PLATINUM), AMLogo avatar small, online dot
   - User can drag divider to resize sections vertically

### LEFT column — three STACKED glass sections (NOT subscription cards here):

Split by TWO draggable dividers (3 rows: store | radio | music):
1. TOP "VIP GOLD" teaser card (~small section):
   - Crown emoji 👑, text "VIP GOLD" lime, "خصم محدود على الباقات"
   - Button "احصل عليه الآن" → opens STORE MODAL (not inline cards)
   - Gift icon button
   - This is a PROMO strip only — NOT full pricing tables
2. MIDDLE "راديو لمة" section:
   - Play/pause, station list, "على الهواء" status, frequency
3. BOTTOM music/DJ section:
   - Owner DJ controls, track list, sync playback

User can drag dividers to resize the 3 left sections.

### CENTER column — CHAT (main hero):

- Room header strip inside chat: room flag + name + welcome topic
- Message list scroll area with glass bubbles
- Own messages: green-tint bubble right; others left with author color + BOSS badge under name for owner
- Bottom composer bar (glass):
  - Rounded input "اكتب رسالتك..."
  - Icons: emoji, gift, mic, image, send orb button (green circle)
- Optional: open room tabs row under header for multi-room
