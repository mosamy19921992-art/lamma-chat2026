## STRIP 1 — Room Topic Bar (~32px min-height)

Full width flex row, items stretch, shrink-0.

### RIGHT part (RTL start) — Topic shell — `flex-1` — class: lamma-topic-shell

Content:
- 📌 pin icon in small rounded button (lamma-topic-pin)
- Topic text OR inline edit input

Default topic per room (examples):
- egypt: "أهلاً وسهلاً بكم في غرفة مصر 💚"
- arab: "مرحباً بكم في ملتقى كل العرب 🌍"
- posts-feed: "انشر أفكارك..."
- admin: "لوحة رقابة المشرفين..."
- owner: "جدار المالك السري 👑..."
- fallback: "مرحباً بكم في الغرفة الذكية ✨"

Behavior:
- Click topic → admin/owner can edit inline (input appears)
- Enter or blur → saves to roomTopics[activeRoomId] state
- Regular users: read-only display

Typography:
- text ~10.5px, font-black, teal-50, truncate single line

Border: border-l border-white/10 separating from ticker

### LEFT part (RTL end) — Entry ticker — FIXED width

Widths: w-[116px] sm:w-[148px] md:w-[188px]
Class: lamma-frost-ticker, border-r border-white/10

Three rotating states (AnimatePresence motion):

**A) join** — someone entered room:
- Green LogIn icon in rounded box
- Nickname (RTL truncate)
- "joined the room" (LTR small text)

**B) leave** — someone left:
- Gray rotated LogIn icon
- Nickname
- "left the room"

**C) standby** (default when no join/leave event):
- Pulsing green dot
- "X متصل" or "متصل واحد"
- "في الغرفة الآن"

Ticker auto-returns to standby after timeout when join/leave events finish.

## STRIP 2 — Room Tabs + Actions (~30px height)

Full width, h-[30px], dir=rtl, px-2, flex justify-between, shrink-0.

### RIGHT side (RTL start) — Open room tabs — `flex-1` overflow-x scroll

Horizontal browser-style tabs from `openRooms` array.

Initial tabs:
- Always starts with egypt tab: { id:"egypt", name:"مصر", flag:"🇪🇬" }
- If user opens via ?room=arab → also tab for arab

When user clicks room in right sidebar:
- If not in openRooms → add tab { id, name, flag }
- Switch activeRoomId

Each tab shows:
- Flag emoji (11px, scale up + gold glow when active)
- Room name (9.5px truncate)
- Active tab extras:
  - Gold gradient text (yellow-300 → amber-200 → yellow-400)
  - layoutId spring animation background (white/10)
  - Small amber ping dot (4px)
  - class: lamma-room-tab-active

Inactive tab: gray-400, lamma-room-tab, hover white

Close button (X):
- On hover of tab (opacity 0 → 100)
- Hidden for egypt tab (cannot close default room)
- Closing active tab → switch to egypt

Tab styling: rounded-t-lg, min-w 70px, pb/pt padding, lamma-room-tab glass gradient

### LEFT side (RTL end) — Utility icon buttons — shrink-0 gap-1.5

Icons in lamma-room-strip-action (~2rem buttons):

1. **Search** (Search icon 14px)
   - Toggle message/member search popover
   - Active: green lamma-quiet-power-btn-active

2. **Plus / Features** (Plus icon, rotates 45° when open)
   - Hidden in posts-feed room
   - Opens dropdown/sheet "الميزات الإضافية":
     - Quick gifts grid (rose, heart, crown...)
     - Achievements block
     - Active call widget (if calling)

3. **Shield** (admin/owner only, hidden in posts-feed)
   - Red tint, opens "الرقابة والأمان" quick panel
   - Ghost mode toggle, etc.
