## HOW TO USE EACH FILE IN CODE (after download)

### 1) login-hero.jpg — login background
```css
.lamma-login-shell {
  min-height: 100vh;
  background:
    radial-gradient(900px 520px at 88% 12%, rgba(255,196,99,0.16), transparent 60%),
    radial-gradient(700px 420px at 8% 18%, rgba(255,120,60,0.10), transparent 55%),
    radial-gradient(800px 480px at 50% 100%, rgba(16,185,129,0.08), transparent 60%),
    url("/images/login-hero.jpg") center / cover no-repeat !important;
}
```

### 2) lamma-app-icon-512.png — gold logo on login screen 1
```html
<img
  src="/images/lamma-app-icon-512.png"
  alt="Lamma Chat"
  width="235"
  style="max-width:66vw; filter:drop-shadow(0 8px 25px rgba(214,179,91,0.75))"
/>
```
Fallback if nice logo fails: `/images/lamma-wordmark.svg`

### 3) MAN.png — chat wallpaper behind glass panels
```css
.lamma-chat-root {
  background: url("/MAN.png") center / cover no-repeat fixed;
}
```

### 4) Favicon + manifest
```html
<link rel="icon" href="/images/lamma-favicon.svg" type="image/svg+xml" />
<link rel="apple-touch-icon" href="/images/lamma-app-icon-180.png" />
```
